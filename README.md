# TreeTraversal
### Tree Traversal in C++

This C++ program reads a pre-order traversal of a non-empty tree and outputs a post-order traversal for the tree.

#### Input: 
{Non_negative_node_value} {Child_count}  
{Non_negative_node_value} {Child_count}  
{Non_negative_node_value} {Child_count}  
Etc...

#### Example:
##### The following input:
3 2  
4 0  
3 4  
2 0  
4 1  
5 0  
6 0  
7 0  

##### Results in the following output:
4 0  
2 0  
5 0  
4 1  
6 0  
7 0  
3 4  
3 2
